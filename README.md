# Obsidian Theme Kimbie Dark

> Welcome to the Kimbie Dark color scheme for Obsidian!

This color scheme is designed to enhance the visual aesthetics and readability of your Obsidian notes. It features a dark, muted color palette inspired by the _Kimbie Dark_ theme, which creates a calm and focused environment for writing and reading.
